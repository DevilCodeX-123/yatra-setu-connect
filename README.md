# Yatra Setu - India's Smart Public Bus Network

Yatra Setu is a comprehensive platform designed to modernize and streamline public bus transportation in India.

## Key Features

- **Real-time Bus Tracking**: Track buses in real-time for better travel planning.
- **Smart Booking**: Easy and secure ticket booking system.
- **Passenger Dashboards**: Personalized dashboards for passengers to manage bookings and profiles.
- **Role-based Access**: Dedicated panels for Admins, Drivers, and Owners.

## Tech Stack

- **Frontend**: React, Vite, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Node.js, Express, MongoDB
- **Authentication**: JWT-based secure authentication with password hashing.

## Getting Started

### Prerequisites

- Node.js (v16+)
- MongoDB Atlas account or local MongoDB instance

### Installation

1. Clone the repository.
2. Install frontend dependencies:
   ```sh
   npm install
   ```
3. Install backend dependencies:
   ```sh
   cd backend
   npm install
   ```
4. Set up environment variables in `.env`.
5. Start the development servers:
   - Frontend: `npm run dev`
   - Backend: `cd backend && npm run dev`
